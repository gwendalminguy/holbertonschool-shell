# Shell

This repository focuses on the use of a [Shell](https://en.wikipedia.org/wiki/Shell_(computing)).

* [Basics](https://github.com/gwendalminguy/holbertonschool-shell/tree/main/basics), a project to learn about using a terminal and a shell.

* [Permission](https://github.com/gwendalminguy/holbertonschool-shell/tree/main/permissions), a project to learn about permissions on Linux-based systems.

* [I/O Redirections & Filters](https://github.com/gwendalminguy/holbertonschool-shell/tree/main/io_redirections_and_filters), a project to learn about using file-related commands, including redirections and filters.

* [Init Files, Variables & Expansions](https://github.com/gwendalminguy/holbertonschool-shell/tree/main/init_files_variables_and_expansions), a project to learn about using environment variables and variable expansion.

* [Automations](https://github.com/gwendalminguy/holbertonschool-shell/tree/main/automations), a project to create simple automations.
